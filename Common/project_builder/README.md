# project_builder

`project_builder` は、DSL形式のテキストからフォルダ、空ファイル、内容付きファイルを生成する Python/tkinter ツールです。

## 特徴

- `dir:` でフォルダを生成します。
- `file:` で空ファイルを生成します。
- `write:` で内容付きファイルを生成します。
- `---` で `write:` ブロックを終了します。
- UTF-8 でファイルを書き込みます。
- Windows での利用を想定しています。
- 相対パスは禁止です。
- 作成ログを GUI に表示します。
- 同名のフォルダまたはファイルが存在する場合はスキップします。

## 起動方法

```powershell
python Common\project_builder\main.py
```

## DSL 仕様

```text
dir: C:\work\sample_project
file: C:\work\sample_project\empty.txt
write: C:\work\sample_project\README.md
# sample_project

Generated by project_builder.
---
```

### コマンド

| コマンド | 内容 |
| --- | --- |
| `dir:` | 指定した絶対パスにフォルダを作成します。 |
| `file:` | 指定した絶対パスに空ファイルを作成します。 |
| `write:` | 指定した絶対パスに内容付きファイルを作成します。 |
| `---` | `write:` の内容入力を終了します。 |

## 注意事項

- すべてのパスは絶対パスで指定してください。
- 既存の同名パスは上書きせず、ログに `SKIP exists` と表示します。
- `file:` と `write:` の親フォルダが存在しない場合は自動作成します。
